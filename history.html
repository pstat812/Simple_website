History of OSI model


Before we get into know the history behind the OSI model , how about we understand why this model exists first?
The OSI model aims to provide a set of standards design for the equipment manufacturers to communicate with the other device, just like we all need a language to talk right?
After people setting this goal , this is where the OSI model beign !

Year Description
1969

First, more and more people had agreed that it is necessary to set up an international standard for packet switching networks so that it can be viable to use for worldwide.

1972

International Network Working Group (INWG) forms to develop an international standard for packet-switching networks.

1974

IBM launches a packet-switching network called the Systems Network Architecture.
Two engineers, Vint Cerf and Robert Kahn, from INWG published "A Protocol for Packet Network Intercommunication," in IEEE Transactions on Communications. (Click here to know more about TCP/IP)

1977

OSI model was committed by International Organization for Standardization (ISO) for packet switching network

1983

OSI model was published as ISO 7498: “The Basic Reference Model for Open Systems Interconnection” as an international standard in May
